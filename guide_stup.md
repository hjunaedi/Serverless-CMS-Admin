# Setup Guide: CMS

1. **Google Sheets**: Create a blank sheet.
2. **Apps Script**: Copy the code from the **Backend Code** tab.
3. **Setup**: Run the `setup` function in Apps Script once.
4. **Deploy**: Deploy as Web App, set access to **Anyone**.
5. **Config**: In this CMS App Settings, paste your Web App URL.
6. **Images**: (Optional) Put your ImageKit keys in Settings and the CONFIG sheet.
